Instalação

Siga os passos abaixo para configurar seu próprio template:

git clone https://github.com/andromarcio/doc-template.git cd doc-template # Abra o index.html no navegador!

Como Usar

Customize o template editando os arquivos HTML, CSS e JavaScript:

  • index.html: Estrutura e conteúdo
  • styles.css: Estilos e temas
  • script.js: Funcionalidades interativas

API & Documentação

Acesse a documentação completa clicando em Framework → global/ para ver os arquivos de referência.

Exemplos

Perguntas Frequentes

Como customizar as cores?

Edite as variáveis CSS em styles.css no :root

Como adicionar novas seções?

Adicione novos elementos <section> no HTML

Como fazer deploy?

Use GitHub Pages habilitando nas configurações do repositório
