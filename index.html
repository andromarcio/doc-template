Instalação

Para começar, clone o repositório:

git clone https://github.com/andromarcio/doc-template.git
cd doc-template
# Abra o arquivo index.html no seu navegador

Como Usar

Edite o arquivo index.html com seu conteúdo:

  1. Atualize o título da documentação
  2. Personalize os itens do menu no sidebar
  3. Adicione suas seções de conteúdo
  4. Customize as cores em styles.css

Perguntas Frequentes

Como customizar o tema?

Edite as variáveis CSS no topo do arquivo styles.css

É responsivo?

Sim! O template é totalmente responsivo e funciona em mobile, tablet e desktop

Preciso de dependências?

Não! Apenas HTML, CSS e JavaScript puro. Font Awesome é opcional para ícones.
